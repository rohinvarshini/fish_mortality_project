Explainable AI-Based Fish Mortality Risk Prediction Using Algal Bloom Image Detection and Water-Quality Fusion

1)Predicts fish mortality risk in aquaculture ponds using Artificial Intelligence and Machine Learning techniques.
2)Detects algal bloom severity from pond-surface images using a CNN-based image classification model (MobileNetV2).
3)Integrates water-quality parameters such as dissolved oxygen (DO), temperature, pH, turbidity, ammonia, and nitrate for comprehensive analysis.
4)Combines image-based bloom detection and sensor-based environmental data through a fusion framework for improved prediction accuracy.
5)Forecasts short-term water-quality trends using a BiLSTM deep learning model.
6)Provides multi-level fish mortality risk classification to support proactive farm management.
7)Uses SHAP-based Explainable AI (XAI) to identify and visualize the key factors influencing predictions.
8)Generates automated risk reports and sends critical-risk alerts to farmers.
9)Helps reduce fish losses, improve water-quality monitoring, and support timely decision-making in aquaculture.
